# QuestXP

A dashboard-first web application featuring gamification, AI-based content processing, video integration, and dynamic study plans.

## Getting Started

1. Clone the repository
2. Install backend dependencies: `cd backend && npm install`
3. Install frontend dependencies: `cd frontend && npm install`
4. Set up `.env` files based on the examples.
5. Run the dev servers: `npm run dev` in both directories.
